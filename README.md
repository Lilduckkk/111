## 特性（Features）

按模块分类展示功能，突出技术亮点：

### 核心能力
- 🚀 **高并发调度**：基于异步IO模型，单节点支持10万+任务/秒调度
- 🔄 **任务依赖管理**：支持DAG（有向无环图）定义任务依赖，自动解析执行顺序
- 🌐 **跨集群协同**：通过分布式一致性协议（Raft）实现多集群任务协同

### 可靠性保障
- 🛡️ **故障自愈**：节点宕机后自动转移任务，任务执行状态持久化（基于MySQL/Redis）
- ⏱️ **超时控制**：支持任务级/步骤级超时设置，自动终止超时任务
- 📊 **全链路监控**：集成Prometheus指标，可视化任务执行延迟、成功率、资源占用

### 易用性设计
- 🖥️ **Web控制台**：提供可视化界面，支持任务创建、启停、日志查看
- 📝 **多语言SDK**：提供Python/Java/Go SDK，5行代码即可接入
- 🧩 **插件扩展**：支持自定义任务类型（如Python脚本、Shell命令、HTTP请求）

## 快速开始（Quick Start）

### 环境要求
- 操作系统：Linux（推荐Ubuntu 20.04+/CentOS 7+）、macOS 12+（开发环境）
- 依赖：
  - Python 3.9+ 或 JDK 11+（根据SDK选择）
  - Redis 6.2+（用于任务队列）
  - MySQL 8.0+（用于元数据存储，可选）

### 安装方式

#### 方式1：Docker Compose（一键部署，推荐新手）
```bash
# 克隆仓库
git clone https://github.com/你的用户名/项目名称.git
cd 项目名称

# 启动所有服务（包含调度器、执行器、Web控制台、依赖组件）
docker-compose up -d

# 查看启动状态
docker-compose ps
